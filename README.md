# A probabilistic view of spatial extreme sea level events in the Baltic Sea

Code and data supporting the article:

> **A probabilistic view of spatial extreme sea level events in the Baltic Sea**  
> Authors: Seuri Basilio Kuosmanen, Magnus Hieronymus  
> Journal: [to appear]

---

## Purpose of this repository

This repository provides the computational material supporting the article, including:

- Reproducibility of all figures and tables presented in the paper
- Model diagnostics and additional analyses provided as supplementary material
- Transparent implementation of the probabilistic models used in the study

The central component of the repository is a Jupyter notebook that serves as the official supplementary material.
